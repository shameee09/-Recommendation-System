🎬 Movie Recommendation System

A Machine Learning-based Movie Recommendation System built using the **MovieLens 100K Dataset**. The project recommends movies using different recommendation techniques, including **Popularity-Based**, **User-Based Collaborative Filtering**, and **Item-Based Collaborative Filtering**.


 📌 Features

- Popularity-Based Movie Recommendation
- User-Based Collaborative Filtering
- Item-Based Collaborative Filtering
- Top-Rated Movie Recommendation
- Genre-Based Recommendation (Action Movies)
- Movie Similarity Analysis

---

📂 Dataset

**MovieLens 100K Dataset**

Files Used:
- `u.data` – User ratings
- `u.item` – Movie information

Dataset contains:
- 100,000 Ratings
- 943 Users
- 1,682 Movies

---
🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Jupyter Notebook


 📊 Recommendation Techniques

⭐ Popularity-Based Recommendation
Recommends the most popular movies based on average user ratings.

👤 User-Based Collaborative Filtering
Recommends movies liked by users with similar rating patterns.

🎥 Item-Based Collaborative Filtering
Recommends movies similar to those previously liked by the user.
